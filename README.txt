# Important

Repo containing app to create .pkg from PlayStation 1 disk images 

## Usage

- Open app RetroXMB_Creator.exe
- Select PSX from options 
- Add a title and select the disk image 
- Click "Make PKG"
- Check everything is right 
- Click "Make PKG" on the bottom right corner of the app

## Note

The result .pkg will be on the Packages folder
